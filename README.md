# zimbra-scanner

install latest python via pyenv
make latest default

```bash
python -m venv ~/venvs/test
. ~/venvs/test/bin/activate
pip install requests
chmod +x ./zimbra_scanner
./zimbra_scanner -i sites.txt
```
